# Summer Institute on Bounded Rationality 2026

Website for the **Summer Institute on Bounded Rationality (SIBR 2026)**, held
8–16 June 2026 at the Max Planck Institute for Human Development, Berlin.

**Live site:** https://valentinkm.github.io/SIBR26/

A static single-page site (`index.html`) that renders the program schedule and
wires each session's abstract and biography from `speakers.json` at runtime.
